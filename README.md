# Smart Email Follow-Up Manager

A full-stack demo application that helps track emails requiring replies and manage follow-ups efficiently.

## Features
- Track incoming emails
- Identify pending replies
- Display follow-up age (days pending)
- Clean dashboard for follow-up management
- Demo mode for safe portfolio presentation

## Tech Stack

### Backend
- Spring Boot
- MySQL
- Spring Data JPA
- Scheduler-based inbox simulation
- REST APIs

### Frontend
- React (Vite)
- Axios
- Responsive dashboard UI
